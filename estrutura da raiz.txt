Dica Importante:
Para testar localmente antes de subir para o servidor:

Use um servidor local como Live Server (VS Code)

Acesse pelo Chrome em https://localhost

No DevTools (F12), vá em Application > Service Workers para verificar o registro

Precisa de ajuda com os ícones ou com o upload para o servidor? Posso te guiar passo a passo!
